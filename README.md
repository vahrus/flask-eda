## Анализ блюд по фото (CLI + веб‑сайт)

Приложение на Python, которое:

- **по CLI** и **через веб‑интерфейс (Flask)** анализирует фотографию блюда с помощью OpenAI (vision, Responses API);
- по желанию отправляет полученный набор продуктов в **GigaChat** и возвращает рецепт нового блюда.

### Установка

1. Перейдите в папку проекта:

```bash
cd "analyze_food"
```

2. (Рекомендуется) создайте и активируйте виртуальное окружение:

```bash
python -m venv .venv
source .venv/bin/activate  # macOS / Linux
# или
.venv\Scripts\activate     # Windows
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Убедитесь, что в файле `.env` заданы необходимые ключи:

```text
OPENAI_API_KEY=ваш_openai_ключ
GIGACHAT_CREDENTIALS=...         # уже есть в примере
GIGACHAT_MODEL=GigaChat          # опционально
GIGACHAT_SCOPE=GIGACHAT_API_PERS # опционально
GIGACHAT_VERIFY_SSL_CERTS=false  # как у вас в примере (для разработки)
```

---

## CLI‑режим

Базовый вызов:

```bash
python main.py path/to/image.jpg
```

Параметры:

- `image` – путь к файлу изображения (PNG, JPG, JPEG, WEBP, GIF);
- `--model` – модель OpenAI для анализа (по умолчанию `gpt-4.1-mini`);
- `--detail` – уровень детализации изображения: `low`, `high`, `auto` (по умолчанию `high`);
- `--gigachat-recipe` – дополнительно отправить анализ блюда в GigaChat и получить рецепт.

Пример:

```bash
python main.py "images/pasta.jpg" --model gpt-4.1-mini --detail high --gigachat-recipe
```

На выходе вы получите:

- структурированный отчёт об анализе блюда от OpenAI;
- (опционально) рецепт нового блюда от GigaChat.

---

## Веб‑интерфейс (Flask)

Запуск веб‑приложения:

```bash
python app.py
```

По умолчанию сайт будет доступен по адресу `http://127.0.0.1:5000/`.

На сайте доступны:

- **поле для загрузки файла** с фотографией блюда;
- **отображение результата работы OpenAI** (структурированный отчёт);
- **кнопка “Получить рецепт от GigaChat”**, которая отправляет текст анализа (с набором продуктов) в GigaChat и показывает рецепт нового блюда;
- **возможность загружать новые файлы** (форма всегда доступна на главной странице).
